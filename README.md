# Оценка надежности пароля

Этот мой учебный проект, который представляет собой простой скрипт на Python. Нужно ввести время боту (цифра с окнчанием s - 5s) и он ответом в сообщении пришлет таймер с прогрессбаром на уменьшение времени.

## Установка

1. Убедитесь, что у вас установлен Python 3.
2. Создайте файл `.env` в корневом каталоге проекта и добавьте следующие переменные среды:
```bash
TOKEN_API = токин тг бота
```
## Использование
1. Скачайте следующие файлы:
    - tg_bot.py - главный файл;
    - ptbot.py - не обходимый учебный файл;
2. Запустите скрипт:
```bash
python ваш_скрипт.py
```
